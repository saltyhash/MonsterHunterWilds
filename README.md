# External Mods Used
- [Hud Controller](https://www.nexusmods.com/monsterhunterwilds/mods/2758) - Amazing UI tweaking and hotkey presets
- [Skip Dialog Boxes](https://www.nexusmods.com/monsterhunterwilds/mods/1902) - Must have for cleaning up stupid extra clicks in the game.
- [Auto-skip Repetitive Cutscenes](https://www.nexusmods.com/monsterhunterwilds/mods/573) - Very nice auto skip (e.g. armor crafting cutscene).
- [Auto-item Pouch Restock](https://www.nexusmods.com/monsterhunterwilds/mods/38) - Restocks your last used item loadout so you can skip menus.

# My own mods
- [What's That Element](https://www.nexusmods.com/monsterhunterwilds/mods/3849) - Displays element type of current weapon (keep your fit consistent)
- [Linked Loadouts](https://www.nexusmods.com/monsterhunterwilds/mods/3708) - Linked item loadouts to armor loadouts - auto loads item loadouts
- [Convenient Rest Menu](https://www.nexusmods.com/monsterhunterwilds/mods/4047) - Adds Rest and Ship Purchase to Nata (fast rest spam)
- [Mute Guitar and Watergun](https://www.nexusmods.com/monsterhunterwilds/mods/3150)
